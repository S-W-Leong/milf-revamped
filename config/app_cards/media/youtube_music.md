# YouTube Music App Guide

## Navigation
- YouTube Music commonly opens to the last used tab, playlist, album, or player. Check the current screen before acting.
- Primary areas are Home, Samples, Explore, Library, Search, and the now-playing player. Use visible labels, content descriptions, and stable resource IDs when available.
- Use Back to leave the full-screen player, queue, album, playlist, artist page, or search results before changing sections.
- Prefer visible text, content descriptions, and stable resource IDs over fixed coordinates.

## Search
- Use the search icon or search field to find a song, artist, album, playlist, or podcast.
- Search with the exact title or artist when provided. If the request is vague, use the most visible matching result but avoid paid, explicit, or user-account changes unless requested.
- If multiple results have similar names, choose the one matching the requested media type, artist, or playlist owner. If still ambiguous, ask for clarification.
- Do not open unrelated web results, YouTube video pages, or external apps unless the user specifically asked for that handoff.

## Playback
- To start playback, open the requested song, album, playlist, artist radio, or podcast and tap the primary play control.
- The mini-player usually appears at the bottom after playback starts. Tap it to open the full player when queue, repeat, shuffle, cast, or lyrics controls are needed.
- Verify the visible title and artist before reporting success.
- Pausing, resuming, skipping, replaying, shuffling, and changing repeat are reversible playback controls and usually do not need confirmation.
- Starting loud media can be disruptive. If the user did not clearly ask to play audio, confirm before starting playback.

## Library and Playlists
- Use Library for saved playlists, albums, songs, artists, podcasts, and downloads.
- When opening a playlist or album, verify the title before tapping Play, Shuffle, or a specific track.
- Adding, removing, downloading, liking, disliking, subscribing, or changing library state modifies the user's account. Use the available confirmation flow before the final tap.
- If a requested playlist, downloaded item, or library entry is not visible, report that it is not visible instead of guessing from similar results.

## Queue and Player Options
- Use the queue or Up Next screen to inspect upcoming tracks, move through the list, or remove queued items.
- Queue edits affect the current listening session. Use confirmation before removing, clearing, or heavily reordering the queue unless the user explicitly requested that exact change.
- Lyrics, comments, related songs, song details, and artist pages are safe to open for inspection.
- Use the overflow menu carefully. Options may include Save, Download, Share, Start radio, Add to playlist, Go to artist, Report, and subscription actions.

## Cast, Devices, and Output
- Cast and device output controls may route audio to speakers, TVs, headphones, or external devices.
- Verify the device name before connecting, disconnecting, or changing output.
- Connecting to an external device can affect other people nearby. Use confirmation immediately before changing cast or output devices.

## Safety
- Do not purchase, subscribe, accept trials, change account settings, clear history, delete downloads, share links, or open external payment/account flows unless the user's request clearly requires it.
- Use the available confirmation flow before any account, billing, subscription, playlist mutation, download deletion, share, cast/output change, or external handoff.
- If YouTube Music shows an ad, age gate, paid feature prompt, sign-in issue, update screen, or permission prompt, stop and report the blocker.

## Recovery
- If playback controls are hidden, tap the mini-player or now-playing area to reveal them.
- If the keyboard covers controls, use Back once to hide the keyboard.
- If search results are stale or empty, clear the search field and retry with the exact media title or artist.
- If the app is playing the wrong item, pause playback, go back to the results, and choose the more specific match.
