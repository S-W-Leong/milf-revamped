# WhatsApp App Guide

## Navigation
- WhatsApp commonly opens to the last used tab or chat. Check the current tab before acting.
- Primary areas are Chats, Updates, Communities, and Calls. Use visible tab labels or accessibility descriptions when available.
- Use the Back button to leave a chat, media viewer, contact profile, or search result before changing tabs.
- Prefer visible text, content descriptions, and stable resource IDs over fixed coordinates.

## Search and Contacts
- Use the search icon from the Chats or Calls screen to find a person, group, or message thread.
- Search by the exact display name when provided. If multiple results appear, choose the one matching the requested person or group.
- If the target is not found, do not guess from similar names. Ask for clarification or report that the contact is not visible.
- Contact profile screens often expose call, video, mute, and media controls; confirm you are on the intended contact before using them.

## Messaging
- Open the correct chat before composing text.
- The message composer is usually at the bottom of the chat screen. Attachments, camera, microphone, emoji, and send controls sit around it.
- After typing, verify the composed text is in the intended chat before sending.
- Sending a message is consequential. Use the available confirmation flow before tapping Send when the user did not already explicitly confirm the final send.

## Voice and Video Calls
- Calls can start from a chat header, contact profile, or Calls tab.
- Use the phone icon for voice calls and the camera/video icon for video calls.
- Verify the displayed contact or group before starting a call.
- Starting a call is consequential. Use the available confirmation flow immediately before tapping the voice-call or video-call control.

## Media and Attachments
- Attachment options may include Gallery, Camera, Document, Audio, Location, Contact, and Poll.
- When sharing media or documents, verify the selected item and destination chat before sending.
- Avoid opening unknown links, payments, or external handoff flows unless the user specifically asked for them.

## Safety
- Do not send messages, start calls, share media, open payment flows, or invite contacts unless the user's request clearly requires it.
- Use the available confirmation flow immediately before any final send, call start, payment, location share, or external handoff.
- If the target person, group, message, or media item is ambiguous, ask for clarification instead of guessing.

## Recovery
- If the keyboard covers controls, use Back once to hide the keyboard.
- If search results are stale or empty, clear the search field and retry with the exact display name.
- If WhatsApp shows a permission prompt, update prompt, blocked account notice, or QR/link-device screen, stop and report the blocker.
